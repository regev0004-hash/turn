Это готовый проект мониторинга очереди на границе Беларусь-ЕС.

Полностью готовый React проект со следующими функциями:

## 🚀 Способ 1: Развертывание на Vercel (РЕКОМЕНДУЕТСЯ)

### Шаг 1: Распакуйте архив

На Windows: двойной клик на архив  
На Mac/Linux: `tar -xzf border-queue-monitor-ГОТОВО.tar.gz`

### Шаг 2: Загрузите на GitHub

Откройте PowerShell/Terminal в папке проекта и выполните:

```bash
git init
git add .
git commit -m "Border queue monitor - ready to deploy"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/border-queue-monitor.git
git push -u origin main
```

**Замените `YOUR_USERNAME` на ваше имя на GitHub!**

### Шаг 3: Подключите Vercel

1. Откройте https://vercel.com/dashboard
2. Нажмите **"Add New Project"**
3. Выберите **"Import Git Repository"**
4. Вставьте: `https://github.com/YOUR_USERNAME/border-queue-monitor`
5. Нажмите **"Continue"**
6. На экране импорта всё оставьте как есть (Vite автоматически обнаружится)
7. Нажмите **"Deploy"**

✅ Через 2-3 минуты приложение будет работать на `https://your-project.vercel.app`

## 🚀 Способ 3: GitHub Pages (встроенный хостинг)

### Шаг 1-2: (как выше - загрузить на GitHub)

### Шаг 3: Включите GitHub Pages

1. На GitHub откройте ваш репозиторий
2. Перейдите в **Settings → Pages**
3. Выберите **"GitHub Actions"** как источник
4. Нажмите **"Create workflow yourself"**
5. Вставьте этот код:

```yaml
name: Deploy

on:
  push:
    branches: [ main ]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
    - uses: actions/checkout@v3
    - uses: actions/setup-node@v3
      with:
        node-version: 24
    - run: npm install
    - run: npm run build
    - uses: peaceiris/actions-gh-pages@v3
      with:
        github_token: ${{ secrets.GITHUB_TOKEN }}
        publish_dir: ./dist
```

6. Нажмите **"Commit"**

✅ Через 2-3 минуты приложение будет работать на `https://YOUR_USERNAME.github.io/border-queue-monitor`

## 🆘 Если что-то не работает

### Ошибка при git push:
```bash
# Используйте Personal Access Token вместо пароля
# Или используйте SSH ключ
```

- Проверьте интернет соединение
- Нажмите кнопку "Обновить"
- Откройте консоль браузера (F12) и посмотрите ошибки

### Vercel/Netlify не развертывает:
- Проверьте логи в панели управления
- Убедитесь что Node.js версия 24.x в package.json
